## 解法一：
### 实现思路：
以中心节点作为对称轴，两边进行互换

### 实现代码
```java
class Solution {
	public void reverseString(char[] s) {
		int right = s.length - 1;
		int left = 0;

		while(left < right) {
			char temp = s[right];
			s[right] = s[left];
			s[left] = temp;
			left++;
			right--;
		}
	}
}
```
